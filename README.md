# 🏈 NFL Scouting Combine Dashboard

Dashboard interativo desenvolvido com **Streamlit** e **Plotly Express** para explorar os dados do NFL Scouting Combine desde o ano 2000.

## Sobre o projeto

O NFL Scouting Combine é um evento anual onde atletas universitários demonstram suas habilidades físicas para os times da NFL antes do Draft. Este dashboard permite explorar o desempenho de mais de 8.500 atletas em testes como corrida de 40 jardas, salto vertical, supino e outros.

## Funcionalidades

- **Filtros dinâmicos** por posição, intervalo de anos e status no Draft
- **Métricas resumidas** (total de atletas, draftados, posições, anos)
- **Histograma** da distribuição do tempo no 40 jardas por posição
- **Gráfico de dispersão** de Peso × Tempo no 40 jardas
- **Tabela interativa** com os dados filtrados

## Tecnologias utilizadas

- Python 3
- Pandas
- Plotly Express
- Streamlit

## Como executar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
